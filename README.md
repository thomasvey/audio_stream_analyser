# Get started
We recommand to create a virtual enviroment with conda.

- create
```bash
    conda env create -f environment.yml
```
- enter env
```bash
    conda activate audio_analyser
```
- list packages
```bash
    conda list
```
- leave env
```bash 
    conda deactivate
```
- update
```bash
    conda env update
```
- remove
```bash
    conda env remove -n audio_analyser
```
